Coloquei esta pasta para armazenar ícones SVG gratuitos das peças de xadrez.
Sugestão: https://commons.wikimedia.org/wiki/Category:SVG_chess_pieces

Os nomes dos arquivos devem ser:
- wK.svg (rei branco)
- wQ.svg (rainha branca)
- wR.svg (torre branca)
- wB.svg (bispo branco)
- wN.svg (cavalo branco)
- wP.svg (peão branco)
- bK.svg (rei preto)
- bQ.svg (rainha preta)
- bR.svg (torre preta)
- bB.svg (bispo preto)
- bN.svg (cavalo preto)
- bP.svg (peão preto)

Após baixar, coloque os SVGs nesta pasta para ativar o tema de peças com ícones.
